Longer Post

Hello world! I’m continuing my slightly anachronistic (or preposterous) tour of Web publishing. I started with a threejs megaproject, and now I’ve travelled all the way back (via simple paperjs attempts) to a static http website. Marvellous.

I think I’ll stay here for a while, build up some foundations, then head back up the ladder of complexity. Maybe one day I will finish my 3d playground!

This microblog is written on my phone in gitjournal, built with Hugo (I still don’t have a build pipeline set up so I run the following script whenever I want to update the site), then published with GitHub pages. Marvellous!

#!/usr/bin/env bash

(
cd journal
git pull
)

(
cd ffettes
rm -rf public/*
hugo -D
)

(
cd ffettes/public
git add .
git commit -am "$1"
git push
)

git add .
git commit -am "$1"
